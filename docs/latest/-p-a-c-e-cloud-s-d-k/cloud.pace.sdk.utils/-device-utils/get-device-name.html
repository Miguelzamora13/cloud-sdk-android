getDeviceName

fun getDeviceName(): String
Content copied to clipboard

Returns the device name formatted the following way: [BRAND or MANUFACTURER] MODEL ([PRODUCT or DEVICE])

Example output:

  • OnePlus ONEPLUS A3003 (OnePlus3)

  • samsung SM-G935F (hero2ltexx)

  • Sony D6603 (D6603)

  • google Nexus 5X (bullhead)

  • Huawei ALE-L21 (ALE-L21)
